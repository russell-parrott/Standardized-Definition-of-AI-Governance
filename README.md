## Overview

This repository contains two formally linked reference standards:

1. **The Standardized Definition of AI Governance (Version 1.0.4)**  
   *The structural definition, the 25-test verification framework, and the three-domain integrity model.*

2. **The AI Governance Measurement Framework (Version 1.0.1)**  
   *The seven-metric solvency system, the loss channels (NTAD, PRE, VAG), and the macro-economic formulation of Annual Governance Loss (AGL).*

Together they form a single doctrine:  
**Definition ‚Üí Tests ‚Üí Metrics ‚Üí Economic Quantification.**  
Structure first, measurement second.

Both documents are immutable, open, and non-derivable under **CC BY-NC-ND 4.0**.

## Structural Logic

The two standards operate as a single system with strict sequencing:

1. **Definition** ‚Äî establishes the structural conditions for governance.  
2. **Tests (25)** ‚Äî binary, adversarial verification of those conditions.  
3. **Metrics (7)** ‚Äî continuous solvency measurement within the same domains.  
4. **Economic Formulation** ‚Äî quantifies trust failure as economic loss.

- Good metrics without passed tests = unverified assumption.  
- Passed tests without metrics = governance decay.
